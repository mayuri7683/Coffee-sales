# Coffee-sales- dashboard
An interactive Excel dashboard to analyze coffee sales trends, top products, and regional performance.
🔍 Features
- **KPIs**: Total sales, average order value, best-selling roast
- **Charts**: Monthly sales, sales by country, top 5 coffee types
- **Slicers**: Filter by roast level, size, loyalty card, or coffee origin
- **Dynamic update**: Add new rows to source → refresh all pivots & charts
| Column | Description |
|--------|-------------|
| Order Date | Transaction date |
| Coffee Type | Arabica, Robusta, etc. |
| Roast | Light / Medium / Dark |
| Size | Small / Medium / Large |
| Unit Price | Price per unit |
| Quantity | Units sold |
| Sales | Total revenue |
| Customer Type | Regular / Loyalty |
| Country | Market region |
